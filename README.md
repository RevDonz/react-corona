## Getting Started

### Installation

1. Clone the repo
```sh
git clone https://github.com/RevDonz/react-laslesvpn.git
```
2. Install NPM packages
```sh
npm install or yarn install
```
3. Run project
```sh
npm run start or yarn start
```
